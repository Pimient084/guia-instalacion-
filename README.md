# Guía de Instalación SquirrelMail

Guía interactiva paso a paso para instalar un servidor de correo con SquirrelMail en una VM Debian/Ubuntu.

## 🚀 Deploy en GitHub Pages

El deploy es automático cada vez que haces push a `main` gracias al workflow de GitHub Actions incluido.

### Pasos para activar GitHub Pages (solo la primera vez):

1. Ve a tu repo en GitHub → **Settings** → **Pages**
2. En **Source**, selecciona **GitHub Actions**
3. Haz push a `main` y el sitio se publica solo

La URL será: `https://<tu-usuario>.github.io/guia-instalacion-/`

## 🛠️ Desarrollo local

```bash
npm install
npm run dev
```
